FAQ

Is this just google forms?

You can do everything this site does with google forms yes. The value prop here is a very minimal setup, a more professional presentation, automated post-event emails, and the ability to export to excel / google sheets.

Can I change the form to add my own questions?

Yes, if you pay for the premium version, which is comming soon. :)

Why is my event in a moderation queue?

I personally review the events to make sure people are not using the site for spam, if you make the events private there is no moderation queue.

Can other people edit the event?

Yes, if you give them the password they should be able to edit the event (presentation lineup and such).
